Crypto Fish Trader

Tools: Python

Inspired by Michael Reeves's YouTube Video, Crypto Fish Trader simulates a fish picking whatever crypto it wants to buy using an API. Note: This fish does not automatically buy crypto, it just tells users what to buy. This program provides a graphic visual if users want to view the fish in action.

Adjust the Counters

Tools: C++

A search algorithm based project using A star algorithm where for this project is to optimally solve the adjust the counters puzzle in the minimal amount of moves possible. The description of the puzzle itself is on the github page.

Kanji Hunt

Tools: Unity, C#

A 3D interactive game where the purpose is to learn the meaning of a Kanji under pressure. The game curently has 50 different Kanjis. The player will have to find a single random Kanji from 10 randomly spawned Kanji picked out by a script from the 50 different Kanjis. Should the player find the correct Kanji within the time constraint, the player will win and receive a score of the time they found the coin.
